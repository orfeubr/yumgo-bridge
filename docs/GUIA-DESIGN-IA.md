## 4️⃣ Animações e GIFs

### 🎬 **Lottie Files** (Recomendado)

**URL:** https://lottiefiles.com

**Buscar por:**
- "food delivery" → Escolher animações de moto, entrega
- "loading food" → Spinners temáticos
- "cooking" → Animações de preparo
- "shopping cart" → Carrinho animado

**Como usar:**
1. Baixar arquivo .json
2. Integrar com biblioteca Lottie: `npm install lottie-web`
3. Usar em Blade:
```blade
<div id="lottie-delivery"></div>
<script src="https://unpkg.com/@lottiefiles/lottie-player@latest/dist/lottie-player.js"></script>
<lottie-player src="/animations/delivery.json" background="transparent" speed="1" loop autoplay></lottie-player>
```

## 📐 Especificações Técnicas

### **Tamanhos de Logo:**
- Logo completo (horizontal): 2000x600px (PNG transparente)
- Logo ícone (quadrado): 1024x1024px (PNG transparente)
- Favicon: 512x512px → converter para .ico

### **Imagens de Produtos:**
- Resolução: 1200x1200px (quadrado)
- Formato: WebP (melhor compressão) ou JPG (qualidade 85%)
- Background: Branco ou transparente (PNG)

### **Banners:**
- Hero Desktop: 1920x600px
- Hero Mobile: 750x1000px
- Instagram Post: 1080x1080px
- Instagram Stories: 1080x1920px

### **Ícones:**
- SVG preferencial (escalável)
- PNG: 256x256px (retina), 128x128px (normal)
- Cor primária: #EA1D2C (vermelho YumGo)

## 🎯 Exemplo de Workflow Completo

### **Dia 1: Identidade Visual**
1. Gerar 10 variações de logo no Leonardo.ai
2. Escolher melhor versão
3. Refinar no Canva (ajustar cores exatas)
4. Exportar em todos os tamanhos
5. Atualizar `/public/images/logo/`

### **Dia 2: Ilustrações**
1. Criar personagem mascote (Leonardo.ai)
2. Gerar 5 ícones de status (Midjourney ou Leonardo)
3. Criar ilustrações de categorias (6-8 itens)
4. Otimizar e exportar como SVG ou PNG

### **Dia 3: Fotos de Produtos**
1. Gerar 15 fotos realistas (Leonardo Kino XL)
2. Processar e otimizar para WebP
3. Upload para `/public/images/products/`
4. Atualizar seed/migrations

### **Dia 4: Marketing**
1. Criar hero banner desktop + mobile
2. Criar templates para Instagram (posts + stories)
3. Criar animações Lottie para loading
4. Integrar tudo no projeto

## 🚀 Dicas Profissionais

### **Para Logos:**
- Teste em preto e branco primeiro (deve funcionar)
- Simples > Complexo (pense na escala de favicon)
- Máximo 2-3 cores
- Evite gradientes complexos (dificulta impressão)

### **Para Ilustrações:**
- Mantenha estilo consistente em todos assets
- Use mesma paleta de cores
- Prefira vetorial (SVG) quando possível
- Sempre peça "white background" ou "transparent background"

### **Para Fotos:**
- Iluminação consistente (todas parecerem do mesmo lugar)
- Mesma perspectiva (top view OU 45 graus)
- Background uniforme (branco puro)
- Alta resolução (4K+) → redimensionar depois

### **Otimização:**
```bash
# Converter PNG para WebP (90% menor)
cwebp input.png -q 85 -o output.webp

# Otimizar PNG
pngquant input.png --quality=85-95 --output=output.png

# Redimensionar em lote
mogrify -resize 1200x1200 -quality 85 *.jpg
```
